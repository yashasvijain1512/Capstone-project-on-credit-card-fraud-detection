# Model Comparison Dashboard

This folder contains the standalone entrypoint for model comparison:
- `model_comparison_dashboard.py`

Run from project root:

```bash
streamlit run Dashboard/model_comparison_dashboard.py
```

The same model-comparison experience is integrated in the main app under `Dashboard Section -> Model Comparison`:

```bash
streamlit run streamlit_app.py
```

For complete feature documentation (metrics, charts, threshold analysis, data input, and quick tour), see the root `README.md`.
